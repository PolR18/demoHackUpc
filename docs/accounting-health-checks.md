# Accounting health checks

## User type
Accountant

## District
Admin Server

## Purpose
Shows which service health checks matter before trusting accounting dashboards.

## What this file contains
- Visits Service must be running.
- Customers Service must be reachable.
- Vets Service must be reachable.
- Gateway must route correctly.

## Agent behavior
Open this file when the accountant asks why a report is incomplete, why dashboard data is missing, or whether the system is healthy.

## Demo trigger phrases
- report incomplete
- missing dashboard data
- service health
- health check
