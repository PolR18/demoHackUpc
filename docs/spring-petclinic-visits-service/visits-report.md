# Opening visits report

## User type
Accountant

## District
Visits Service

## Purpose
Shows a finance-friendly summary of visit activity across clinics.

## What this file contains
- Total visits created in the selected period.
- Completed, cancelled, and pending visits.
- Visits that may require billing review.
- Clinics or vets with unusual activity.

## Agent behavior
Open this file when the accountant asks about reports, visit summaries, billable visits, or monthly activity.

## Demo trigger phrases
- opening visits report
- show visit report
- monthly visits
- billable visits
