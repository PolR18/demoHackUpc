# Reviewing vet performance summary

## User type
Accountant

## District
Vets Service

## Purpose
Provides a high-level financial and operational view of veterinarian activity.

## What this file contains
- Total visits handled.
- Average visits per week.
- Specialty coverage.
- Clinic assignment.
- Activity compared with previous period.

## Agent behavior
Open this file when the accountant asks about vet activity, performance summaries, or workload analytics.

## Demo trigger phrases
- vet performance
- vet summary
- workload analytics
