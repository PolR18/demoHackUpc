# Accounting configuration overview

## User type
Accountant

## District
Config Server

## Purpose
Explains configuration that can affect accounting dashboards and reports.

## What this file contains
- Active profile.
- Service URLs.
- Report feature flags.
- Dashboard refresh interval.
- Environment-specific settings.

## Agent behavior
Open this file when the accountant asks why numbers differ between environments, why a report option is enabled, or where accounting configuration lives.

## Demo trigger phrases
- config
- feature flag
- environment
- report settings
