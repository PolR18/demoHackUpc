# Visit dashboard wireframe

## User type
Designer

## District
Visits Service

## Purpose
Describes the UI structure for a visit activity dashboard.

## What this file contains
- Header with selected clinic and date range.
- KPI cards for total, completed, cancelled, and pending visits.
- Main chart showing visits over time.
- Table with recent visits.
- Empty state for clinics with no activity.

## Agent behavior
Open this file when the designer asks about the visit dashboard, KPI cards, layout, or visual hierarchy.
