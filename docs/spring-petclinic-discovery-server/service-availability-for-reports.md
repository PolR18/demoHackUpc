# Service availability for reports

## User type
Accountant

## District
Discovery Server

## Purpose
Explains how service discovery affects reporting data availability.

## What this file contains
- Required services must be registered.
- Missing services can hide dashboard sections.
- Gateway depends on discovery to route correctly.

## Agent behavior
Open this file when the accountant asks why report data is missing, why a service cannot be found, or how services are located.

## Demo trigger phrases
- service unavailable
- missing data
- discovery
- service registry
