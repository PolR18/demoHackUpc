# Service discovery visualization

## User type
Designer

## District
Discovery Server

## Purpose
Explains how to visually represent services appearing or disappearing from the system.

## What this file contains
- Live service registry list.
- Animated service nodes.
- Warning badge when a service is unavailable.
- Connection lines between gateway and registered services.

## Agent behavior
Open this file when the designer asks about service discovery, live system maps, availability, or animated infrastructure views.

## Demo trigger phrases
- service discovery
- live map
- registry
- availability visualization
