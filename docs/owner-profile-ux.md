# Owner profile UX

## User type
Designer

## District
Customers Service

## Purpose
Describes the experience for an owner profile page.

## What this file contains
- Owner name and contact information.
- Linked pets.
- Recent visits.
- Alerts for missing profile information.
- Actions: edit owner, view pet, review visits.

## Agent behavior
Open this file when the designer asks about owner profiles, customer profile screens, or pet-owner relationships.

## Demo trigger phrases
- owner profile
- customer profile
